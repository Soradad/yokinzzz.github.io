数据结构与程序设计-读书笔记

Feb 28, 2016 | -Learning

　　
从开始看这本书开始，已经接近半年没有写代码（好惭愧，对于C++的各种基本语法都有些生疏.刚看完栈的介绍，于是写了一道习题，熟悉一下栈的方法和C++基本语法. 

#include<iostream>  
#include<stack>  
using namespace std;  
int main()
{
    stack<char>front_symbol;  
    char symbol;  
    bool is_matched = true;  
    while(is_matched && (symbol = cin.get()) != '\n'){  
        if(symbol == '[' || symbol == '{' || symbol == '(')  
               front_symbol.push(symbol);  
        if(symbol == ']' || symbol == '}' || symbol == ')'){  
            if(front_symbol.empty()){  
                cout << "Unmatched closing bracket" << symbol <<   "detected" <<endl;  
                is_matched = false;  
           }  
            else{  
                char match;  
                match = front_symbol.top();  
                front_symbol.pop();  
                is_matched = (symbol == ')' && match == '('  
                              ||symbol == ']' && match == '['  
                              || symbol == '}' && match == '{');  
                if(!is_matched)  
                    cout << "Bad match" << match << symbol << endl;  
            }  
        }      
    }     
    if(!front_symbol.empty())  
        cout << "Unmatched opening bracket detected" << endl;  
}

　　栈作为最简单的数据结构之一，看完书后基本能有一个比较清晰地认识.由于栈这个数据结构比较简单，所以结构本身没有什么值得记下来的地方.但是在自己实现栈这个数据结构的时候，一定要注意栈的溢出，要做一定的异常处理及反馈.
